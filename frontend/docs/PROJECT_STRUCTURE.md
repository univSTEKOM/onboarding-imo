## Where do I put a new …?

| New thing | Goes in |
|---|---|
| Page / URL | `src/routes/` (file-based; run dev to regenerate `routeTree.gen.ts`) |
| API call for a resource | `src/lib/services/<resource>.service.ts` |
| Server data hook | `src/hooks/use-<resource>.ts` (wrap `useQuery`/`useAppMutation`) |
| Client UI state | a Zustand store in `src/lib/stores/` |
| Form / search-param schema | `src/lib/schemas/` |
| A modal, table columns, cards | `src/components/features/<resource>/` |
| Shared chrome (layout-level) | `src/components/templates/` |
| A tiny reusable widget | `src/components/ui/` |

> **Never hand-edit `routeTree.gen.ts`.** It is regenerated by the TanStack Router
> Vite plugin whenever route files change.
